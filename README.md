## Mô tả từng bài

### Bài 01 - Profile Card
- Thẻ giới thiệu bản thân có ảnh avatar, họ tên, ngành học, kỹ năng, sở thích
- Nút chuyển đổi chế độ sáng/tối bằng `classList.toggle()`
- Animation khi card xuất hiện, hiệu ứng hover

### Bài 02 - Landing Page (Quán Cà Phê Mộc)
- Dùng Bootstrap 5 CDN
- Có Navbar responsive, Hero section, 6 card thực đơn, section About, Contact
- Smooth scroll khi bấm menu, navbar đổi màu khi cuộn trang

### Bài 03 - Form Validation
- Form đăng ký có: họ tên, email, số điện thoại, giới tính, mật khẩu, checkbox
- Kiểm tra từng trường bằng JavaScript: `event.preventDefault()`, `value.trim()`
- Nút hiện/ẩn mật khẩu, hiển thị thông báo thành công sau khi hợp lệ

### Bài 04 - List Search & Filter
- Mảng dữ liệu 9 khóa học với tên, danh mục, mô tả, giá
- Tìm kiếm theo tên (realtime), lọc theo danh mục, sắp xếp theo giá
- Dùng `filter()`, `includes()`, `sort()` trong JavaScript

### Bài 05 - Mini Website (Portfolio)
- Website portfolio cá nhân đầy đủ: Navbar, Hero, About, Projects, Skills, Contact
- Dark mode toggle, lọc dự án theo danh mục, smooth scroll, back-to-top button
- Form liên hệ có validation, skill bar animation

## Những gì tôi đã học

- Cách tổ chức file HTML, CSS, JavaScript theo cấu trúc thư mục rõ ràng
- Dùng Bootstrap 5 để tạo layout responsive nhanh chóng
- Xử lý DOM với JavaScript: `querySelector`, `addEventListener`, `classList`
- Kiểm tra và validate form bằng JavaScript thuần
- Tạo chức năng tìm kiếm, lọc, sắp xếp dữ liệu với `filter()` và `sort()`
- Deploy website lên GitHub Pages
